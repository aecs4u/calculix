About

CalculiX Rust is a comprehensive migration of the CalculiX finite element solver from legacy C/Fortran to modern Rust. This project provides a type-safe, memory-safe, and high-performance FEA solver with a clean modular architecture.

The solver supports multiple element types (truss, beam, shell, solid), linear and nonlinear analysis, modal analysis, and dynamic time integration. Integration with PETSc provides access to industry-leading solver libraries.

Quick Start

Analyze a CalculiX INP file:

cargo run --bin ccx-cli analyze examples/beam/cantilever.inp

Run the solver:

cargo run --bin ccx-cli solve tests/fixtures/solver/beamcr4.inp

Run validation tests:

pytest tests/solver/ -v
